# insert

  easy element insertion

## Installation

  Install with [component(1)](http://component.io):

    $ component install eivindfjeldstad/insert

## API
### .before(el, after)
Insert `el` before `after`.

### .after(el, before)
Insert `el` after `before`.

### .prepend(el, parent)
Prepend `el` to `parent`.

### .append(el, parent)
Append `el` to `parent`.
